# Contribution Guidelines

Thank you for wanting to contribute to **Awesome Vibe Coding**! 

## How to Contribute

### Adding a Resource

1. **Fork** this repository
2. **Add your resource** to the appropriate section in `README.md`
3. **Submit a Pull Request** with a clear title and description

### Format

Please follow this format for new entries:

```markdown
- [Resource Name](https://example.com) - A concise description that explains what makes this resource useful. Max 1-2 sentences.
```

### Quality Standards

Before submitting, please ensure your resource meets these criteria:

-  **Relevant** - Directly related to vibe coding, AI-assisted development, or AI coding tools
-  **Active** - The project/resource is actively maintained (updated within the last 6 months)
-  **Useful** - It provides genuine value to developers interested in vibe coding
-  **Accessible** - The link works and the resource is accessible (free or clearly marked as paid)
-  **Not a duplicate** - Check that it's not already on the list

### What We Don't Accept

-  Self-promotional links with no community value
-  Abandoned or dead projects
-  Resources behind hard paywalls with no free tier
-  Low-quality or AI-generated spam content
-  Tools that are unrelated to AI-assisted coding

## Suggesting a New Category

If you think the list needs a new category, please:

1. Open an **Issue** explaining why the category is needed
2. Include at least 3-5 resources that would fit in the new category
3. We'll discuss it as a community before adding

## Reporting Issues

Found a broken link or incorrect information? Please:

1. Open an **Issue** with the title: `[Broken Link] Resource Name` or `[Correction] Resource Name`
2. Include the current link and, if possible, the correct/updated link

## Code of Conduct

Please be respectful and constructive. See our [Code of Conduct](CODE_OF_CONDUCT.md) for details.

## Thank You! 🙏

Every contribution helps the vibe coding community discover better tools and resources. Your PR helps developers build amazing things with AI.
